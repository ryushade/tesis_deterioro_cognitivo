# Estructura de la app móvil

Esta app está centrada en capturar la foto de un dibujo realizado en la prueba del reloj.

## Estructura
- `App.tsx`: Pantalla principal para capturar la foto.
- `types.d.ts`: Tipos para la foto capturada.
- `assets/`: Carpeta para imágenes y recursos.

## Requisitos
- Permisos de cámara
- Expo ImagePicker (o React Native ImagePicker)

## Próximos pasos
- Guardar la foto localmente o enviarla al backend.
- Añadir autenticación si es necesario.
- Mejorar la UI y agregar validaciones.
