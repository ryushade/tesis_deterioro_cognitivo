# Mobile App

Esta carpeta está destinada para el desarrollo de una pequeña app multiplataforma (iOS y Android).

## Tecnologías sugeridas
- React Native (TypeScript)
- Expo (opcional, para facilitar el desarrollo y pruebas)

## Estructura inicial sugerida
- `src/` - Código fuente de la app
- `assets/` - Imágenes y recursos
- `README.md` - Documentación de la app móvil

Puedes iniciar el proyecto con React Native o Expo dentro de esta carpeta.

---

Para comenzar con Expo:

```sh
npx create-expo-app .
```

Para comenzar con React Native CLI:

```sh
npx react-native init MyMobileApp
```

Luego, conecta la app móvil con el backend Flask usando las APIs REST ya existentes.
