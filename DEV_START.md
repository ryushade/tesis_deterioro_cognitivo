Ejecución de Frontend y Backend juntos
--------------------------------------

Desde la carpeta raíz del repositorio:

- Python directo: `python dev_all.py`
- Windows PowerShell: `./start-all.ps1`
- Vía npm (raíz): `npm run dev:all`

Esto levanta:
- Backend Flask en `http://localhost:5000`
- Frontend Vite en `http://localhost:5173`

Detener: presiona Ctrl+C para cerrar ambos procesos.
